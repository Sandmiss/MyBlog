hexo搭建个人博客

安装

安装前提

  1. node.js
  2. git

安装Hexo

建站

执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件

_config.ym

配置网站信息

配置

  • 格式：

    覆盖主题配置

    在_config.ym中

命令

  1. new

    新建一篇文章,如果没有设置 layout 的话，默认使用 _config.yml 中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。

  2. 清除缓存文件 (db.json) 和已生成的静态文件 (public)。

  3. 生成静态文件。

  4. 启动服务器。默认情况下，访问网址为： http://localhost:4000/。

  5. 部署网站。

  2. 配置github账户信息

    打开SSH
